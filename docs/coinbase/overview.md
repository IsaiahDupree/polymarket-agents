# Advanced Trade API — Overview

> Source: https://docs.cdp.coinbase.com/coinbase-app/advanced-trade-apis/overview
> Mirrored: 2026-05-25

**Platform Overview**

"The Advanced Trade API (or Advanced API) supports programmatic trading and order management with a REST API and WebSocket protocol for real-time market data."

## What It Offers

Advanced Trade functions as a sophisticated trading interface designed for experienced traders, providing capabilities to buy, sell, and trade digital assets across multiple trading pairs.

## Available SDKs

The platform offers several software development kits:
- Official Python implementation
- TypeScript sample
- Go sample
- Java sample
